# SafeWorld (安全世界)
- 一款为创造服打造的插件，禁止玩家熊服等各种功能  

# Q&A
**这个插件只能在主世界使用？**  
- 不止主世界，当然，是你能使用指令或多世界插件去到其他维度的前提下，因为所有原版跨纬度的传送门都已禁用。

**为什么没有防苦力怕的/原版不是有指令禁止TNT爆炸吗？**  
- 对，原版在gamerule里就可以直接禁用生物破坏以及TNT爆炸，你可以直接忽略掉这个插件提供的这项功能。

**凋零是在搭建时被阻止放置还是在搭建好后被阻止？**  
- 是在搭建好后被阻止，你搭建的只是一个凋零的框架，他不会真正被激活，这对那些爱造东西的玩家是一件好事。

**为什么要禁用潜影盒的使用？**
- 因为有些模组在玩家为创造时提供物品NBT编辑等功能所以就导致玩家利用此来刷32K，大部分32K都存储于潜影盒中。如果仍想使用潜影盒，请使用1.0.0版本（不推荐）。

**为什么要推出这么一款插件？**  
- 这个插件是专门给那些仅提供大厅服务或某些创造服使用而诞生，它们通常会禁止玩家做一些破坏服务器的事，也会禁止玩家进入其他维度（原版）。

**为什么要限制掉落物/生物以及高频红石等的功能？**
- 因为有些人会利用此行为卡服，造成服务器崩溃，你随时可以在config.yml里更改。

**为什么配置文件改了之后有些功能还是无法使用？**
- 尽量重启服务器，且部分功能还在测试中，欢迎提出Issue。
