# SafeWorld (安全世界)
- 一款能够保护你独个世界的插件，插件禁止玩家点燃TNT、引爆水晶/重生锚、激活凋灵、末影人搬砖、动物吃草，禁止进入末地与下界  

# Q&A
**这个插件只能在主世界使用？**  
- 不止主世界，当然，你能使用指令或多世界插件去到其他维度的前提下，所有原版跨纬度的传送门都已禁用。  
**为什么没有防苦力怕的/原版不是有指令禁止TNT爆炸吗？**  
- 对，原版在gamerule里就可以直接禁用生物破坏以及TNT爆炸，你可以直接忽略掉这个插件提供的这项功能。  
**凋零是在搭建时被阻止放置还是在搭建好后被阻止？**  
- 是在搭建好后被阻止，你搭建的只是一个凋零的框架，他不会真正被激活，这对那些爱造东西的玩家是一件好事。  
**为什么要推出这么一款插件？**  
- 这个插件是专门给那些仅提供大厅服务或某些创造服使用而诞生，它们通常会禁止玩家做一些破坏服务器的事，也会禁止玩家进入其他维度（原版）。
